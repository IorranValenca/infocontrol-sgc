# InfoControl - Sistema de Gestão Comercial

## 📌 Descrição
O InfoControl é um sistema de gestão comercial desenvolvido para uma loja de informática, com o objetivo de controlar clientes, produtos, vendas e relatórios.

## 🎯 Objetivo
Automatizar os processos comerciais da loja, garantindo organização, controle de estoque e registro eficiente de vendas.

## 🛠 Tecnologias Utilizadas
- Python
- Django
- Django REST Framework
- JWT (JSON Web Token)
- PostgreSQL

## ⚙️ Funcionalidades
- Cadastro de clientes
- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Autenticação de usuários
- Relatórios de vendas

## 🔐 Segurança
- Autenticação via JWT
- Controle de acesso por perfil (ADMIN, FUNCIONARIO)
- Senhas criptografadas

## 🗃 Banco de Dados
O sistema possui as seguintes tabelas:
- usuarios
- clientes
- produtos
- vendas
- itens_venda
